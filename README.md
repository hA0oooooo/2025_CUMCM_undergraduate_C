# 2025_CUMCM_undergraduate_C

**Author: Haoyang Chen, Ruihan Xu, Yuhang Xu** (in alphabetical order)

问题一聚焦于检测指标随孕周与体型变化的规律识别，难点在于样本既包含群体层面的差异，也包含同一受试者的重复检测记录。该问题可拆解为总体趋势刻画与个体差异刻画两部分：前者需要识别孕周、BMI 的线性与非线性影响，后者需要反映不同受试者在基线水平和变化斜率上的差异。为此，先对检测指标进行变换，再构建同时包含固定效应与随机效应的混合效应模型，在统一框架下同时利用群体样本信息和重复测量信息。结果表明，检测指标与孕周呈显著正相关、与 BMI 呈显著负相关，模型在原始指标尺度上的拟合优度约为 0.8830，组内相关系数约为 0.8365，说明个体差异是影响检测结果的重要因素。

问题二聚焦于在不同 BMI 条件下确定较优检测时点，本质是在“尽早检测”与“达到足够检测准确率”之间进行权衡。问题可拆解为三步：先估计给定孕周和 BMI 下达到检测阈值的概率，再构造过早或过晚检测的时间风险代价，最后在连续 BMI 区间上同时优化分组边界与各组推荐时点。实现中采用带概率约束的分段优化思路，通过离散化搜索与动态规划寻找满足覆盖率要求的 5 组方案，并在结果端加入保守校正以避免推荐时点过于激进。结果显示，5 组 BMI 区间的推荐孕周大致从 15.8 周上升到 22.5 周，且各组平均覆盖率均高于 0.95；敏感性分析进一步表明，当可靠性要求提高或误差放大系数增大时，平均推荐时点会稳定后移。

问题三在问题二基础上进一步引入年龄、怀孕次数和生产次数等协变量，形成更细粒度的分层检测时点推荐方案。该问题可拆解为年龄分层、BMI 区间分段和协变量偏移校正三个子问题：先按可解释的年龄区间进行分层，再在每个年龄层内独立完成 BMI 分段优化，最后通过受约束的偏移项吸收协变量对推荐时点的联合作用，并用单调约束保证方案的稳定性与可解释性。实现结果给出了完整的 5×5 推荐表（5 个年龄层 × 每层 5 个 BMI 分段），加权平均推荐孕周约为 14.52 周；同时在误差放大或覆盖率要求提高时，推荐时点呈现可解释的延后趋势，例如当覆盖率要求从 0.95 提高到 0.99 时，加权平均推荐孕周约增加 4.67 周。

问题四聚焦于女胎样本的多类别异常判定，难点在于类别不平衡明显且存在复合异常组合。该问题可拆解为目标编码、多层判定和结果评估三个环节：先将三条染色体异常状态组合编码为 8 类目标，再构建由多个专家分类器与元分类器组成的 Stacking 集成学习判定流程，最后在固定类别顺序下输出二分类与八分类指标及混淆矩阵。实现中结合样本重采样、特征筛选与多模型融合提升少数类识别稳定性，并使用固定五折划分方式完成评估。结果显示，五折评估下总体准确率约为 0.9901，正常/异常二分类 F1 约为 0.9545，八分类宏平均 F1 约为 0.9490；多数类别识别效果较好，但极少数类别的召回率仍存在波动。
